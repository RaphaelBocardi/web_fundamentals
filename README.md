# Web Fundamentals

This repository contains my progress in Web Development, focusing on HTML5, CSS3, and eventually JavaScript. These exercises are part of the Computer Engineering curriculum at UNIVESP.

## 🚀 Technologies
- **HTML5:** Semantic structure and web accessibility.
- **CSS3:** Styling, layouts, and responsiveness (In Progress).
- **IDE:** PyCharm Professional.
- **OS:** Zorin OS (Linux).

## 📂 Repository Structure
- `/html_basics`: Introductory tags, lists, and images.
- `/css_styling`: Styling exercises, colors, and fonts.
- `/projects`: Small static pages like "Virtual Store".

## 🛠️ How to View
To see any page, simply clone the repository and open the `index.html` file in any modern web browser.
